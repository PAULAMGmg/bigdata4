FUENTES DE DATOS

Los datos pueden clasificarse en tres tipos:
      - Estructurados: Datos organizados en tablas (por ejemplo, bases de datos relacionales).
      - No Estructurados: Datos que no tienen una estructura predefinida (por ejemplo, correos electrónicos, documentos de texto).
      - Semiestructurados: Datos que tienen cierta estructura, como archivos JSON o XML
